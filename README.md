# Caspian Workforce Group Website

Production-ready Next.js website for Caspian Workforce Group.

## Verified build

The project was tested successfully with:

```bash
npm ci
npm run build
```

## Run locally

```bash
npm ci
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Upload all files and folders from this project to the root of your GitHub repository.
2. In Vercel, choose **Add New → Project**.
3. Import the GitHub repository.
4. Keep the detected framework as **Next.js** and click **Deploy**.
5. Add `caspianworkforcegroup.com` under **Project Settings → Domains**.

## Forms

The forms currently open a prepared email addressed to `info@caspianworkforcegroup.com`.
A direct server-side email integration can be added later.
